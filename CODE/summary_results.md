# Podsumowanie wyników analizy

Poniżej przedstawiono syntetyczne podsumowanie wyników analizy danych z ankiety dotyczącej przerw w pracy programistów.

| Kalkulacja | Wynik | Wniosek / interpretacja |
|---|---|---|
| Mediana i IQR uciążliwości przerw | Mediana = 3, IQR = 1 | Przerwy są postrzegane jako umiarkowanie uciążliwe; odpowiedzi są dość spójne |
| Mediana i IQR produktywności | Mediana = 4, IQR = 1 | Respondenci deklarują raczej wysoką produktywność |
| Mediana i IQR satysfakcji z pracy | Mediana = 4, IQR = 0 | Satysfakcja z pracy jest wysoka i bardzo jednorodna w próbie |
| Rozkład odpowiedzi – uciążliwość przerw | Najwięcej odpowiedzi: 3 i 4 | Brak silnych skrajności; przerwy są „normalnym” elementem pracy |
| Rozkład odpowiedzi – produktywność | Dominują wartości 3–4 | Większość badanych ocenia swoją produktywność pozytywnie |
| Rozkład odpowiedzi – satysfakcja | Dominują wartości 4–5 | Bardzo wysoki poziom zadowolenia z pracy |
| Korelacja Spearmana: uciążliwość ↔ produktywność | ρ = 0.088, p = 0.469 | Brak istotnej zależności między uciążliwością przerw a produktywnością |
| Korelacja Spearmana: częstotliwość przerw ↔ uciążliwość | ρ = 0.271, p = 0.023 | Im częstsze przerwy, tym są one postrzegane jako bardziej uciążliwe |
| Korelacja Spearmana: czas powrotu do skupienia ↔ uciążliwość | ρ = 0.304, p = 0.010 | Dłuższy powrót do skupienia wiąże się z większą uciążliwością przerw |
| Test Manna–Whitneya: Remote vs On-site | p = 0.565 | Brak istotnych różnic w uciążliwości przerw między trybami pracy |

---

**Jednozdaniowe podsumowanie:**

Przerwy są umiarkowanie uciążliwe, nie obniżają bezpośrednio produktywności ani satysfakcji, natomiast ich częstotliwość oraz koszt poznawczy związany z powrotem do skupienia mają istotne znaczenie.
